### 背诵 20 个新单词

1. **memorize**
  - vt. 记住，熟记，背熟
  - memor·ize => memor- 记忆 + -ize 使 → 记住。

2. **recite**
  - v. 背诵，朗读，叙述
  - re·cit·e => re- 向后，往回 + cite 召唤，援引。引申词义想起，背诵，朗诵。

3. **abandon**
  - vt. 放弃，抛弃，遗弃，沉溺于，放纵
  - n. 放任，放纵
  - a·band·on => a- = ad- + band- 命令 + -on 表名词 → 在别人的控制中，无限制的 → 放弃。

4. **discard**
  - vt. 丢弃，抛弃
  - vi. 垫牌
  - n. 垫牌，抛弃
  - dis-去掉 + card【卡片】→ 把【卡片】扔掉 → 抛弃。

5. **sacrifice**
  - n. 牺牲，供奉，祭品
  - vt. 牺牲，祭祀，贱卖
  - vi. 献祭
  - sacr·i·fic·e => sacr- 神圣的 + -i- + fic- 做 + -e → 做出来献给神的东西 → 牺牲品。

6. **disposable**
  - a. 一次性的，可任意处理的
  - dis·pos·able => dispose【处理】+ -able 表形容词。

7. **cooperate**
  - vi. 合作，协力，配合
  - co·oper·ate => co- 共同 + oper- 工作 + -ate 表动词 → 共同工作。

8. **cooperation**
  - n. 合作，协力，配合
  - co·oper·ation => cooperate【合作】+ -ion 名词后缀，是单词 cooperate 派生的名词。

9. **collective**
  - a. 集体的，聚集的，共同的
  - col·lect·ive => collect【搜集】+ -ive 表形容词 → 集中到一起 → 集体的，是单词 collect 派生的形容词。

10. **collaborative**
  - n. 合作的
  - col·labor·at·ive => col- 一起 + labor- 劳动 + -at + ive 形容词后缀 → 一起劳动的 → 合作的。

11. **joint**
  - n. 关节，接头，连接处，一块烤肉，公共场所，大麻烟卷
  - a. 联合的，共同的
  - v. 把……切成带骨的大块肉
  - joint- = join 结合，连接。

12. **mutual**
  - a. 相互的，共有的
  - mut·ual => mut- 改变，交换 + -ual 形容词后缀 → 你变我也变 → 相互的。

13. **alliance**
  - n. 联盟，联合
  - all·i·ance => ally【使联盟】+ -i- + -ance 表名词 。

14. **defend**
  - vt. 防护，辩护，防卫
  - de·fend => de- 去掉 + fend- 打击 → 去掉打击 → 打退敌人 → 保卫。

15. **defendant**
  - n. 被告
  - de·fend·ant => defend【防护】+ -ant 名词后缀。

16. **defense**
  - n. 防卫，防卫物
  - de·fens·e => de- 去掉 + fens- 打击 + -e → 防止打击 → 防卫。

17. **confront**
  - vt. 使面对，对抗，遭遇，使对质，比较
  - con·front => con- 互相，共同 + front- 面对 → 互相面对 → 对抗。

18. **confrontation**
  - n. 对抗，冲突
  - con·front·ation => confront【v. 对抗】+ -ation 表名词，是单词 confront 派生的名词。

19. **resisant**
  - n. 抵抗者，反抗者
  - a. 抵抗的，反抗的
  - re·sist·ant => re- 反 + sist- 站 + -ant 名词、形容词后缀 → 反着站的人 → 反抗者。

20. **resistance**
  - n. 抵抗力，反抗，耐力，阻力，电阻
  - re·sist·ance => resist【反抗，抑制】+ -ance 表名词。


### 赏析一句英语句子

But unlike their absolutist counterparts in the Gulf and Asia, most royal families have survived because they allow voters to avoid the difficult search for a non-controversial but respected public figure.

但是与海湾地区和亚洲的专制君主不同，大多数欧洲王室能够幸存下来，是因为他们使选民无须费力寻找一个没有争议却受人敬重的公众人物。

#### 句子主干

- But ... most royal families have survived （主+谓）

#### 一级修饰和二级修饰

- **most ... suvived**
  - unlike their absolutist counterparts
  - 介词短语作句子主干的比较状语
  - **their absolutist counterparts**
    - in the Gulf and Asia
    - 介词短语作后置定语，修饰 their absolutist counterparts
- **most ... suvived**
  - because they allow voters to avoid the difficult search
  - because 引导的原因状语从句，作句子主干的原因状语
  - **the difficult search**
    - for a non-controversial but respected public figure.
    - 介词短语作后置定语，修饰 the difficult search

#### 词汇注释

- `unlike` prep. （表示属性）与……不像，与……不同，非……的特点 a. 不同的，相异的，不相似的
- `absolutist` n. 专制主义者
- `counterpart` n. 职位相当的人
- `royal` a. 王室的，皇家的
- `voter` n. 选举人，投票人，有投票权者
- `avoid` vt. 避免，避开，躲避，消除
- `controversial` a. 有争议的
- `respected` a. 受尊敬的
- `figure` n. 人物

### 写一篇知乎回答贴之编程能够带来食物和水吗？

1. 导言：

读完我这篇文章或许能让你颠覆认知，亦或许能让你深受启发，也或者你也有和我一样的想法……
最近在回顾《黑客帝国》前三部，准备看第四部，你这个问题突然激发了我一些思考，觉得蛮有意思的，我来谈谈我的想法。
看完影片，你或许会发出相同的疑问：

**我们这个世界是真实的吗？或者说什么才是真实的？**

![黑客帝国4预告](../../../img/pic2022/jan/d18/theMatrix.jpg)

2. 剧情背景：

在黑客帝国中，“人类”是被种植出来的，而非生育出来的。

在机器文明与人类文明的对抗中，人类在溃败之际实施了「遮蔽天空计划」，以此削弱以太阳能为能源的机器文明。
在机器文明失去了「太阳能」的能量来源后，他们发现人类身体潜藏着许多能量，机器文明建立了大规模的「人类农场」，人类一出生，就被安置在像子宫一样的培养容器中，身上插满了线路。
为了保证人类可以持续为机器供电，机器文明制造了一个「虚拟的人类社会」。
通过数据线，人类大脑被接入「母体——Matrix」中，母体控制着这些被种植出来的人类意识，让他们以为自己生活在1999年的人类社会之中，实际上他们的身体已经变成了为机器文明供电的“电池”。

3. 编程能够带来食物和水吗？

出生并活在虚拟世界中的人类并不知道自己所处的世界只是母体为他们创造的虚拟世界，他们认为自己就是生存在真实世界之中。

在虚拟世界的人类他们能感受到一切感觉，他们能感受到饿，吃了“食物”能够解除饥饿；他们能感觉到渴，喝了“水”能够解渴；他们能感受到疼，能呼吸“新鲜空气”，能感受到世界的美好……
然而这一切只不过是「母体」为这些活在容器内的人类的大脑输送的信号罢了。

男主角「Neo——尼奥」是虚拟世界中的一名软件工程师，同时也是一名黑客。
软件工程师的身份是他在虚拟世界中的真实身份，如同普通人一样安分守己地活在母体创造的虚拟世界中。
在虚拟世界的网络世界中他扮演的黑客，通过不法手段来牟利，也因此不断走向真相。

身为黑客的 Neo 越是在“法律边缘”试探，就越来越感觉到自己所以为的“真实”并非真实。【影片的故事也就此开始……】

我就不剧透影片的内容了，如果对此感兴趣，可以去补一补。

最近非常火的【元宇宙】就是来自《黑客帝国》，如果不了解什么是元宇宙，那么看完《黑客帝国》相比会有一个比较深刻地认知。
当然，「元宇宙」的设计自然没有《黑客帝国》中那么真实，因为我们活在真实世界里，我们并非机器种植出来的人类。

我想我会当程序员，也是因为小时候看了《黑客帝国》这部电影，那个时候在我内心深处种下了种子。
近来回顾的时候，发现这部电影依旧是那么震撼。
身为程序员的我非常清楚地知道，我可以通过“免费的手段”获取现实世界中的水和食物。
技术是把双刃剑，用得好则能给人类带来更美好的生活，用得不好则会扰乱社会秩序甚至给自己带来牢狱之灾。

虽然不能通过不法手段来获取现实生活中的食物和水，但是我依旧相信未来能实现这样一种技术：通过像《黑客帝国》一样借助电脑、编程、脑传感器向大脑发送信号，让人感受真实世界中所有的感觉，如同活在真实世界之中。

人类的能量来源是糖，我们可以给沉浸在虚拟世界的人注射葡萄糖维持身体机能，计算机能处理大脑发送的一切信息，计算机处理后响应传递给大脑。
这种方式或许将超越现在所定义的【元宇宙】，因为他比元宇宙更真实，更具沉浸感，能够构建完全如同《黑客帝国》中母体所构建的虚拟世界。
唯一的区别是，这项技术是人类掌控的而非机器掌控。

我想象到这项技术如果能够落地，会给未来医疗领域带来颠覆性的影响。
比如给植物人使用这项技术，那么可能在某种程度上能够唤醒植物人。
即便不能，那么也能让植物人继续感受真实世界，让真实世界的亲人进入虚拟世界中与植物人沟通交流……

因为前段时间看了《圆桌派》中，尹烨谈「生命科学」这部分内容，看完后久久不能缓过神来，信息量太大了，他的一番论述不断冲击着我对「生命」的限有认知。
或许未来这项技术不单用在植物人身上，而是能够用在生命科学上，人类真正实现与细胞、病毒沟通，从而实现平衡的世界，这或许是「癌症」的最佳治愈手段，或者说世界上的癌症不治而愈。

当然还有很多我没想到的，如果你看我以上的论述，也有什么类似的想法，欢迎评论留言给我。

4. 编程确实能够带来食物和水：

看完以上论述，诚然，编程确实在某种程度、某些方式给你带来食物和水，但这真的是你想要的吗？
不论是非法之地，还是虚拟现实，编程确实能给你带来食物和水。
但我也说过，编程技术应该是让人类生活更美好的一种方式，而非谋取暴利，扰乱秩序的工具。

### 使用 Neovim 自带的 man 查看器

之前我一直在想，如何用 Vim 来查看 `man (Manual)` 文档，但是我找了许久，都没找到相关的插件。
今天在群里看到了一个群友问了这个问题，刚好有人抛了一条 `Stack Overflow` 的相关链接。
里面有很多解决方案，比如：

- **临时文件查看法**
  - 借助 Linux 管道将 man 帮助文档 pipe 到 Vim 中
  - 这种方式的缺点：
    - 无高亮显示
    - 必须强制退出
  - 适用场景：保存 man 文档内容的情景
  - 例子：
    - `man ls | vi`

- **调用内置 Man 文档查看器法**
  - 在 Neovim 中通过使用 `:Man` 命令调用内置 Man 文档查看器
  - 缺点：
    - 无法在终端下直接打开，必须要启动 Vim/Neovim
  - 优点：
    - 有高亮显示，更易读
    - 不是写入到临时缓冲区，不必强制退出
  - 例子：
    - `:Man printf`
    - `:Man 3 printf`

[在不使用临时文件的情况下阅读 `man pages`](https://stackoverflow.com/questions/16740246/what-is-a-way-to-read-man-pages-in-vim-without-using-temporary-files)
