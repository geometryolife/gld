### 背诵 20 个新单词

1. **nutrient**
  - a. 营养的，滋养的
  - n. 营养物，滋养物
  - nutri·ent => nutri- 滋养 + -ent 表形容词，引申词义营养物，营养品。

2. **foster**
  - a. 收养的，养育的
  - vt. 养育，抚养，培养，鼓励，抱（希望）
  - 来自 food 喂养，食物

3. **steal**
  - v. 偷，窃取
  - n. 偷窃，便宜货，（欺诈的）政治交易
  - 来自古英语 stelan 偷，盗窃，来自 Proto-Germanic stelan 偷，来自 PIE stel 偷，抢，词源同 stalk。

4. **hijack**
  - vt. 抢劫，劫持，敲诈
  - hi·jack => hi- 缩写自 highway【公路，干道】，jack 人名 Jack，代指土匪无赖，后词义通用化。

5. **corruption**
  - n. 腐败，堕落，贪污，单词变体
  - cor·rupt·ion => corrupt【腐败】+ -ion 名词后缀，是单词 corrupt 派生的名词。

6. **corrupt**
  - a. 腐败的，贪污的，讹误充斥的
  - vt. 使腐烂，腐蚀，使恶化
  - vi. 腐烂，堕落
  - cor·rupt => cor- 全部 + rupt- 断 → 全部断 → 变坏的，腐败的。

7. **bribery**
  - n. 贿赂行为，行贿，受贿
  - brib·ery => brib- 贿赂 + -ery 表名词 → 贿赂。
  - bribe => 在古英语指施舍给乞丐的小片面包，词源同 break 分开，小块。

8. **gambler**
  - n. 赌博者，赌徒
  - gambl·er => gamble【赌博】+ -er 表人，是单词 gamble 派生的动作执行者（施动）名词。
  - gamble 来自 game 游戏，打赌。

9. **entertainment**
  - n. 娱乐，款待，娱乐表演
  - enter·tain·ment => entertain【招待，维持，娱乐】+ -ment 表名词。
  - enter·tain => enter- 在中间 + tain- 拿住 → 在[工作]中间拿 → 娱乐。

10. **entertaining**
  - a. 使人愉快的，有趣的
  - enter·tain·ing => entertain【招待，维持，娱乐】+ -ing 表形容词 → 娱乐是有趣的。

11. **leisure**
  - n. 空闲，闲暇，悠闲
  - a. 空闲的，有闲的
  - 来自法语 leisir 来自拉丁语 licere 许可，允许，词源同 licit，license。引申词义许可做自己的事，有空，空闲，休闲。词义演变比较 school，sport。

12. **concert**
  - n. 音乐会，和谐，一致，联合行动
  - v. 共同协定，协力，协调，协同行动
  - con·cert => con- 强调 + cert- 区分。

13. **relieve**
  - vt. 减轻，救济，解除，使免除，换……的班，使得到调剂，使不单调，衬托，使显著
  - vi. 救济，当替补投手，呈鲜明突出
  - re·liev·e => re- 再 + liev- 轻 + -e → 再轻 → 减轻。

14. **relief**
  - n. 减轻，解除，救济，安慰，调剂，浮雕，换班，（地势的）起伏
  - re·lief => re- 再 + lief- = liev- 举起，提高，变轻 → 减轻。

15. **alleviate**
  - vt. 减轻，使缓和
  - allev·i·ate => al- 同 ad- 表强调+ lev- 轻 + -i- + -ate 动词后缀 → 使轻松 → 缓和。

16. **alleviation**
  - n. 减轻，缓和，镇痛物
  - al·lev·i·at·ion => alleviate【vt. 减轻】+ -ion 表名词，是单词 alleviate 派生的名词。

17. **addicted**
  - a. 沉溺于……的，无法摆脱（某种嗜好）的
  - ad·dict·ed => addict【使沉溺】+ -ed 形容词后缀。
  - ad·dict => ad- 一再 + dict- 说 → 说了还说 → 沉溺。

18. **addiction**
  - n. 入迷，瘾
  - ad·dict·ion => addict【使上瘾】+ -ion 名词后缀，是单词 addict 派生的名词。

19. **indulge**
  - vt. 纵情于，使高兴，放任，迁就
  - vi. 沉溺
  - in·dulg·e => 回构自 indulgence 爱好，纵容（in- 进入，使…… + dulg- 长的，长时间的）。
  - dulg- 不变的，固定的。

20. **mean**
  - a. 低劣的，卑贱的，简陋的，吝啬的，惭愧的，平均的，中间的，普通的
  - vt. 意谓，想要，意欲，预定
  - vi. 用意，有意义
  - n. 平均数，中间，中庸


### 赏析一句英语句子

Princes and princesses have day-jobs and ride bicycles, not horses (or helicopters).

王子和公主有日常工作，他们骑自行车而不骑马（或搭乘直升机）。

#### 句子主干

- Princes and princesses have day-jobs （主+谓+宾）
- **have day-jobs**
  - and ride bicycles,
  - and 连接的并列谓宾

#### 一级修饰

- **ride bicycles,**
  - not horses (or helicopters).
  - 补充说明
