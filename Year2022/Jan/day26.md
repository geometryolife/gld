### 背诵 20 个新单词

1. **biased**
  - a. 有偏见的
  - bias·ed => bias【偏见】 + -ed 形容词后缀。

2. **bias**
  - n. 偏见，斜纹
  - a. 偏斜的
  - ad. 偏斜
  - vt. 使有偏见

3. **dense**
  - a. 密集的，浓厚的
  - dens·e => dens- 变浓厚 + -e → 浓厚的。

4. **density**
  - n. 密度
  - dens·ity => dense【浓厚的】+ -ity 表名词 → 密度。

5. **character**
  - n. 个性，字符，人物，性质，品格，资格
  - vt. 印，刻，使具有特征
  - char·acter => char- 字符 + -acter 表名词 → 环境在人的身心留下的印记 → 个性。

6. **characteristic**
  - n. 特性，特征，特色
  - a. 特性的，特有的，有特色的
  - char·acter·ist·ic => character【特征】+ -ist + ic 表形容词 → 有特性的，有特征的。

7. **characterize**
  - vt. 是……的特征，以……为特征，描述（人或物）的特性，描绘
  - char·acter·ize => character【特征】+ -ize 表动词 → 描绘，描述特征。

8. **particular**
  - n. 一项（或条、点），个别项目，详细说明
  - a. 特别的，独有的，挑剔的，详尽的
  - part·ic·ular => part- 部分 + -ic + -ular 有……的 → 有自己的部分 → 特别的。

9. **feature**
  - n. 面孔的一部分（如眼、口等），特征，容貌，特色，特写
  - vt. 是……的特色，特写，放映
  - vi. 起重要作用
  - feat·ure => feat- 做，制作 + -ure 名词后缀 → 做事的状态 → 做事特征。

10. **attribute**
  - n. 属性，标志，定语
  - vt. 把……归于，认为……属于
  - at·tribut·e => at- 加强动作 + tribut- 部落，贡品，给予 + -e → [把原因]给出 → 归因。

11. **trait**
  - n. 特征，特性，一笔，少许

12. **temper**
  - n. 脾气，心情，中和剂，倾向，回火，（钢、铁等的）硬度
  - vt. 使回火，锻炼，调和，使缓和
  - vi. 回火
  - temper- = moderate 表示“调和”。

13. **difficult**
  - a. 困难的
  - dif·fic·ult => dif- 不 + fic- 做 + -ult → 不好做 → 困难的。

14. **complex**
  - n. 综合体，情节，络合物
  - a. 复杂的，组合的
  - com·plex => com- 共同 + plex- 折叠 → 全部重叠 → 复杂的。

15. **complexity**
  - n. 复杂，复杂性，复杂的事物
  - com·plex·ity => complex【复杂的】+ -ity 表名词。

16. **complicated**
  - vt. 弄复杂，使错综，使恶化
  - vi. 变复杂
  - com·plic·at·ed => com- 全部 + plic- 折叠 + -ate 表动词 → 全部重叠起来 → 弄复杂。

17. **intricacy**
  - n. 纷乱，复杂，错综
  - in·tric·acy => in- 进入 + tric- 琐事，繁琐 + -acy 表名词 → 进入琐碎之事。

18. **intricate**
  - a. 复杂的，错综的，缠结的，难懂的
  - in·tric·ate => in- 进入，使 + tric- 琐事，繁琐 + -ate 表形容词，即卷入琐事的，引申词义错综复杂的。

19. **reduce**
  - vt. 减少，分解，降低，使衰退，把……分解，把……归纳
  - vi. 减少，减肥，缩小
  - re·duc·e => re- 回 + duc- 引导 + -e → 往回引 → 减少。

20. **reduction**
  - n. 减少，缩影，变化
  - re·duct·ion => re- 回 + duct- 引导 + -ion 名词后缀。


### 赏析一句英语句子

It is only the Queen who has preserved the monarchy's reputation with her rather ordinary (if will-heeled) granny style.

只有女王以她相当平凡的（尽管穿着相当考究的）奶奶风格保住了君主的声誉。

#### 句子主干

- only the Queen...has preserved the monarchy's reputation （主+谓+宾）
- **the Queen**
  - It is...who
  - It is...who 为强调句型，强调主语 the Queen

#### 一级修饰

- **only the...monarchy's reputation**
  - with her rather ordinary (if well-heeled) granny style.
  - 介词短语作句子主干的方式状语

#### 词汇注释

- `preserved` a. 保藏的，腌制的，喝醉的
- `reputation` n. 名气，名声，名誉
- `rather` ad. 相当，宁可，宁愿
- `ordinary` a. 平凡的，普通的，平常的 n. 普通，平常的人（或事）
- `well-heeled` a. 穿着考究的，富有的
- `granny` a. 老太太式样的
